# 🍏 怪物彈珠 - 果實分配助手 (Monster Strike Fruit Allocator)

### 這是一個專為《怪物彈珠》玩家設計的網頁小工具，旨在協助玩家有效管理角色的英雄果實分配、計算庫

### 存需求，並追蹤刷果實的進度。

## ✨ 主要功能

### 1. 👥 角色管理

### 新增/刪除角色：建立需要安裝果實的角色清單。

### 搜尋功能：快速尋找特定角色，支援關鍵字過濾。

### 清單管理：提供獨立的彈窗介面來檢視與刪除角色。

### 2. 🎒 果實庫存盤點

### 庫存追蹤：記錄目前持有的果實數量。

### 智能計算：系統會根據分配表自動計算「已使用量」，並即時顯示庫存狀態：

### 充足 (✓)：庫存剛好或有剩餘。

### 剩餘 (+)：顯示多餘的數量。

### 短缺 (-)：以紅色顯示缺少的數量，方便補貨。

### 自訂分類：除了預設的加擊類（同族/戰型/擊種）與常見果實（熱友/將命/兵命/速必）外，支援使用

### 者自行新增類別與果實名稱。

### 3. ⚖ 分配與進度追蹤

### 果實配置：為每個角色分配最多 4 顆果實。

### 進度勾選：配置後可勾選「已獲得」，方便辨識還有哪些角色需要刷果。

```
快速組合 (Presets)：提供一鍵套用常用組合功能：
同族加擊全套 (加擊/加命擊/加擊速)
戰型加擊全套
擊種加擊全套
速必雙削組合 (將命/兵命/速必)
進階篩選：
僅顯示搜尋結果：專注於特定角色或果實的分配。
隱藏已完成：自動隱藏果實已全數畢業的角色，讓介面更清爽。
```
4. 💾 資料保存
    自動儲存：所有操作皆會即時儲存於瀏覽器的 LocalStorage，關閉分頁後資料不遺失。
    備份與還原：
       匯出：將當前所有資料打包下載為 .json 檔案（檔名包含日期）。
       匯入：讀取備份檔以還原資料，方便跨裝置傳輸或備份進度。

## 🛠 技術架構

### HTML5: 使用語意化標籤建構清晰的頁面結構。

### CSS3:

```
CSS Variables: 統一管理主題色（綠、藍、紅）與漸層背景。
Responsive Web Design (RWD): 支援桌面與行動裝置，手機版表格會自動轉為卡片式佈局。
Glassmorphism: 採用現代化的毛玻璃視覺風格與動態背景。
JavaScript (Vanilla):
無依賴任何外部框架 (No jQuery, No React/Vue)。
模組化的函式設計 (Render, Storage, Event Handling)。
使用 FileReader API 處理檔案匯入。
```
## 🚀 如何使用

### 1. 開啟網頁

### 本專案為純靜態網頁，無需架設後端伺服器。

```
直接用瀏覽器打開 index.html 即可開始使用。
```
2. 建立角色
    在「角色管理」區塊輸入角色名稱並點擊「新增」。
3. 盤點庫存 (選用)
    在「果實庫存」區塊輸入你背包現有的果實數量。
4. 分配果實
    捲動至下方「分配表」找到該角色。
    使用下拉選單選擇計畫安裝的果實，或使用上方的「快速組合」按鈕一鍵套用。
    觀察庫存區塊的數字變化，確認是否缺貨。
5. 追蹤進度
    當你在遊戲中確實幫角色裝上該果實後，勾選該欄位旁的核取方塊 (Checkbox)。
